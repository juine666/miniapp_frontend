# 快速测试清单 ⚡

## 🚀 启动服务

```bash
# 1. 启动后端（新终端窗口）
cd miniapp-backend
./gradlew bootRun -x test

# 等待看到：Started MiniappBackendApplication in X.XXX seconds
```

然后在微信开发者工具中打开 `miniapp-frontend` 项目。

---

## ✅ 测试步骤（按顺序执行）

### 1️⃣ 登录 (2分钟)
- [ ] 打开小程序
- [ ] 点击底部"我的"标签
- [ ] 点击头像或"点击登录"
- [ ] 授权登录（开发者工具会自动模拟）
- [ ] ✅ 验证：看到头像、昵称、"设置"和"退出"按钮

### 2️⃣ 发布商品 (3分钟)
- [ ] 点击底部"发布"标签
- [ ] 点击图片区域，选择一张图片
- [ ] ✅ 验证：图片上传后立即显示在页面
- [ ] 填写标题：例如 "测试商品123"
- [ ] 填写价格：例如 "99"
- [ ] 填写描述：例如 "这是一个测试商品"
- [ ] 点击"发布闲置"
- [ ] ✅ 验证：看到"已发布"提示
- [ ] 返回"我的"页面
- [ ] 点击"我发布的"标签
- [ ] ✅ 验证：能看到刚发布的商品

### 3️⃣ 收藏商品 (1分钟)
- [ ] 点击底部"首页"标签
- [ ] ✅ 验证：能看到商品列表（包括刚发布的）
- [ ] 点击任意商品右上角的 🤍 按钮
- [ ] ✅ 验证：按钮变成 ❤️，看到"已收藏"提示
- [ ] 点击"我的" → "收藏"（统计数据）
- [ ] ✅ 验证：在收藏页面看到收藏的商品

### 4️⃣ 查看详情 (1分钟)
- [ ] 在首页点击任意商品卡片
- [ ] ✅ 验证：进入详情页，显示完整商品信息
- [ ] ✅ 验证：能看到价格、描述、卖家信息
- [ ] ✅ 验证：底部有"和卖家聊一聊"和"我想要"按钮

### 5️⃣ 购买流程 (2分钟)
- [ ] 在详情页点击"我想要"按钮
- [ ] ✅ 验证：订单创建成功（可能弹出支付）
- [ ] 返回"我的"页面
- [ ] 点击"我买到的"标签
- [ ] ✅ 验证：能看到刚创建的订单

### 6️⃣ 测试分享 (1分钟)
- [ ] 在首页商品列表，点击商品右上角的 📤 按钮
- [ ] ✅ 验证：跳转到分享页面
- [ ] ✅ 验证：分享页面显示商品图片和信息
- [ ] 点击页面任意位置
- [ ] ✅ 验证：跳转到商品详情页

## 🐛 常见问题快速解决

| 问题 | 解决方案 |
|------|---------|
| 图片上传后不显示 | 查看控制台，检查URL是否正确；刷新页面 |
| 发布失败 | 检查是否填写了标题和价格；查看后端日志 |
| 登录失败 | 在开发者工具中是正常的（使用mock模式） |
| 收藏/订单看不到 | 检查是否已登录；查看"我的"页面的统计数据 |
| 后端连接失败 | 确认后端服务已启动（`http://localhost:8081`） |
